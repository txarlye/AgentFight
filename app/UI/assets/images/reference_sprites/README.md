# Sprites de Referencia - brawler_tut

Estos sprites fueron descargados del repositorio [brawler_tut](https://github.com/russs123/brawler_tut) 
para ser utilizados como referencia en la generación de sprites para SimpleFight v3.

## Archivos incluidos:
- `Idle.png` - Sprite de reposo
- `Walk.png` - Sprite de caminata
- `Attack.png` - Sprite de ataque
- `Block.png` - Sprite de bloqueo
- `Hurt.png` - Sprite de daño
- `Jump.png` - Sprite de salto

## Uso:
Estos sprites se utilizan como referencia para el agente de generación de sprites
para mantener un estilo consistente con juegos de lucha pixel art.
